# step_template

Prerequisites

- Docker is up and running
- Git installed

# Deploy an environment for a single use

```
git clone --recursive https://github.com/4-DS/step_template.git
cd step_template
```

## To make use of it, run:
```
bash run_sinara.sh
```

## To stop using it for a while, run:
```
bash stop_sinara.sh
```

## To continue using it, run:
```
bash run_sinara.sh
```
